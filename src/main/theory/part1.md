# 1. 모던 자바
## 모던 자바란?
* JDK 8 버전 이후의 자바를 의미
* 새로운 기능이 많이 추가됨

### 왜 배워야 하는가?
> 아니 할 것도 많은데, 굳이 현대문법까지?
* 코드가 더 짧고, 읽기 쉬워지고, 유지보수하기 좋아짐
* 병렬 처리를 쉽게 만들 수 있음
* 함수형 프로그래밍 패러다임의 이해
* **미래의 기술을 익히는데 도움이 된다는 뜻!**

### 무엇을 배우는가?
* 람다식
* 스트림
* 컬렉션의 추가기능
* Optional 클래스
* 자바 모듈
* 비동기 프로그래밍
* 함수형 프로그래밍

## 자바 함수
* 모던 자바에서는 함수도 파라미터로 쓸 수 있다.
* 그럼 뭐가 좋은데? 더 간결해짐.
* 어떻게 쓰면 돼?
  1. 메서드 참조
  2. 람다 표현식

## 스트림
* 반복문이 너무 길면 어떻게 할까?
* 마치 터미널의 '파이프'처럼 꼬리물기를 하면서 이걸 해결할 수 있다.
* 이를 스트림(stream)이라고 한다.
